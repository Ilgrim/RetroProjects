INTRODUCTION

Quijote can read an html file and show its contents on the screen.

Text justification, bold, underline and italic are supported (but no graphics, sorry).

It runs under Samaruc, my GUI for the Amstrad PCW.

USE

In the CP/M command line, enter:

A> QJT filename.htm

to browse the html file, or:

A> QJT

to enter to the Quijote main menu.

By example: QJT QJT.HTM

By default, no mouse is selected (KEYBOARD EMULATION). If you have one (AMX, KEMPSTON or KEYMOUSE), enter to the Samaruc Control Panel, and select the right model.
